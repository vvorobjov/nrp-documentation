How to debug my experiment
==========================

.. todo:: Add author/responsible


This tutorial guides you through steps to debug a faulty experiment. Upon completing the tutorial,
you will have learnt about the data monitors, the visualizers, the logging mechanism, the online
editing tools and you will have fixed the tutorial example experiment.

Your initial move is to connect to the online Platform from http://neurorobotics.net if you are
a registered user, or to start your locally installed Platform.

Then you need to browse the template experiments and clone the one called **Tutorial: How to debug
my experiment**. You can use the filter to find it easily.

Finally, start the simulation by hitting the **Launch** button and follow Step 1 hereunder.
